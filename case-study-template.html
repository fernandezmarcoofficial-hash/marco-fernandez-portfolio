System Security Audit Tool

Building smarter security automation with Linux scripting and compliance monitoring.

Overview

This project automates Linux system audits, generating detailed reports on vulnerabilities, user permissions, and package integrity. It’s built to assist cybersecurity teams in maintaining compliance effortlessly.

Goals

  • Automate manual audit tasks using Bash scripting
  • Generate real-time system integrity reports
  • Enhance security posture with proactive monitoring

Process

The project began with identifying repetitive audit processes in IT departments. I then mapped each check into automated functions with organized log outputs and version control integration.

Technologies Used

Bash Linux (Mint) GitHub VS Code

Key Results

Reduced manual audit time by 80%. Increased transparency in system change tracking. Improved team confidence in compliance readiness and endpoint hardening.
